# Objetivo

Desenvolver um modelo de aprendizado de máquina capaz de auxiliar na identificação de padrões relacionados ao sono humano utilizando dados disponibilizados pelo Human Sleep Project (BDSP-HSP).

O objetivo específico consiste em construir um classificador capaz de distinguir a presença ou ausência de determinada condição clínica relacionada ao sono (ou, conforme o conjunto efetivamente utilizado, classificar o tipo de atividade/repouso).

# Definição da tarefa

Tipo de problema:

Classificação Binária

Atributo-alvo:

Indicador de Apneia do Sono (ou variável *kind*, caso permaneça o conjunto simplificado).

Atributos preditivos:

- idade
- sexo
- eficiência do sono
- índice de despertares
- frequência cardíaca
- demais atributos clínicos disponíveis
